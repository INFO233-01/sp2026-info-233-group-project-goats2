# Group Project Phase I

INFO-233 Group Project – Phase I – Project Definition
Project Name: NBA Stats Program
Team Name: GOATS2

1.	Define the problem you are trying to solve. - It is difficult to gather stats for Nba Players
2.	How does your program help to solve the problem?  - It creates a program that can gather all the players stats in seconds
3.	Who will be the users of this program. Detail at least two Use Cases 
a.	Use Case 1 - Your debating basketball players and you need quick access to their stats to prove your point
b.	Use Case 2 - A commentator needs to able to check the stats of the players during the game
4.	What technology is available to solve the problem?  Identify the specific APIs and Modules to be used for the program.
- The Free Nba Hall of Fame API from Rapid API allows us to pull the stats, our modules are Import Requests and Import Randoms
6.	Detail the sequence of the program- What is the flow of the program? How does a user use the program. 
- the program starts and it requests the user to input the name of a Nba Player
- Then using the Api it pulls the information and informs the user about the career stats of the player
- This includes their Points, Rebounds, Assists along with their Teams and Positions.
